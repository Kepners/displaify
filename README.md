# Displaify

**Turn your tablet into a second display for Windows** - connect via WiFi or USB.

## Features

- **Extend your display** - Use your tablet as a second monitor
- **WiFi or USB** - Wireless convenience or wired low-latency
- **Lenovo tablet support** - Tested and optimized
- **Touch passthrough** - Interact with Windows from your tablet
